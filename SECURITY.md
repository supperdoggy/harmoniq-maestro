# Security Policy

## Supported Scope

Security fixes are prioritized for:

- `album-queue`
- `spotdl-wapper`
- `dynamic-playlists`
- `models`

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report via private channel:

- Email: `security@supperdoggy.dev`

Include:

- Affected component and version/commit
- Reproduction steps
- Impact assessment
- Suggested mitigation (if available)

## Response Targets

- Initial response: within 3 business days
- Triage status update: within 7 business days
- Fix timeline: depends on severity and complexity

## Secrets Policy

Any exposed credentials in repository history should be considered compromised,
rotated, and revoked before public release.
